Detailed Description

Calculates the time difference between all detectors and a reference detector.

  • The timestamps and timeshifts are in picoseconds Goal 1 : calculate the time shifts between detectors How : Use one detector as reference, preferencially with good time resolution Create coincidences using the CoincBuilder event builder Calculate the difference in timestamp as follow : ref_time - detector_time Finally, output the (.dT) file containing the time shifts of all the detectors Also produces a .root file containing the raw and corrected time spectra. This module includes multithreading management using MTObject::parallelise_function(function, parameters...) Goal 2 : read a timeshift file (.dT) previously calculated by this module Goal 3 : apply the timeshifts by calling the operator[] : time_correct = time_shifted + timeshifts[label]; Example at the end
