Detailed Description

Class used to read .fast files.

This class is to be used in combination with the Hit struct Here is the minimal code you can use to scan the files : 

 Hit hit;
 std::string filename;
 FasterReader reader(&hit, filename);

 while(reader.Read())
 {
   print(hit);
 }

This method deals with the 5 following members of the Hit structure: 

   struct Hit
   {
     unsigned short label;  // The label number of the detector
     Float_t        nrj;    // The ADC/QDC value of the hit. For RF, contains the frequency.
     Float_t        nrj2;   // The QDC2 value (usually the longer gate)
     ULong64_t      time;   // High precision time in ps (time tick of 7,8125 ps)
     bool           pileup; // Contains either pileup or saturated bit
   }

-------— Define section -------—

Some options can be activated at compilation time. You have to #define them before including this library. e.g. : 

   ... other includes
   #define FASTER_GROUP
   #include <FasterReader.hpp>
   rest of the code....

– QDC1MAX

By default the nrj2 is handled. If no detector uses the QDC2 then declare it

– FASTER_GROUP

If the data is grouped using a hardware trigger then the reading is done in two steps : first extracts all the hits of the group and put it in a vector, then each call of Read() moves in the group. No group information is extracted : from outside the class everything goes as if there was no group
