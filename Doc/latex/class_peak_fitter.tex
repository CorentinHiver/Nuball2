\doxysubsection{Detailed Description}
Allows one to fit a peak of a histogram in the range [low\+\_\+edge, high\+\_\+edge]. 

\begin{DoxyAttention}{Attention}
The edges must be well centered, this is not a peak finder. 
\end{DoxyAttention}
