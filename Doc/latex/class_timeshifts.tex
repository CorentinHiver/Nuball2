\doxysubsection{Detailed Description}
Calculates the time difference between all detectors and a reference detector. 


\begin{DoxyItemize}
\item The timestamps and timeshifts are in picoseconds Goal 1 \+: calculate the time shifts between detectors How \+: Use one detector as reference, preferencially with good time resolution Create coincidences using the \doxyref{Coinc\+Builder}{p.}{class_coinc_builder} event builder Calculate the difference in timestamp as follow \+: ref\+\_\+time -\/ detector\+\_\+time Finally, output the (.dT) file containing the time shifts of all the detectors Also produces a .root file containing the raw and corrected time spectra. This module includes multithreading management using MTObject\+::parallelise\+\_\+function(function, parameters...) Goal 2 \+: read a timeshift file (.dT) previously calculated by this module Goal 3 \+: apply the timeshifts by calling the operator[] \+: time\+\_\+correct = time\+\_\+shifted + timeshifts[label]; Example at the end 
\end{DoxyItemize}
