\doxysubsection{Detailed Description}
EXPERIMENTAL Composed of a \doxyref{Path}{p.}{class_path} and a \doxyref{Filename}{p.}{class_filename}. 

A \doxyref{File}{p.}{class_file} object is composed of a \doxyref{Path}{p.}{class_path} and a \doxyref{Filename}{p.}{class_filename} object, which are composed of \+:
\begin{DoxyItemize}
\item A list of folder that forms the \doxyref{Path}{p.}{class_path} to the file
\item A short name and an extension for the \doxyref{Filename}{p.}{class_filename} \begin{DoxyRefDesc}{Todo}

\end{DoxyItemize}
