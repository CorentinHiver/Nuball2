\doxysubsection{Detailed Description}
Loads the ID file and fills lookup tables. 

Many lookup tables are created \+: ... Detectors\+::index(\+Label label) \+: Returns the detector\textquotesingle{}s index in its type. For instance, if there are 10 germaniums, and their labels ranges contiguously from 10 to 19, Detectors\+::index(12) = 2;

Detectors\+::type(\+Label label) \+: Returns the detector\textquotesingle{}s type knowing its label. Taking previous example, Detectors\+::type(25) = \char`\"{}null\char`\"{} if there are no detector above label = 19, and Detectors\+::type(13) = \char`\"{}ge\char`\"{};
