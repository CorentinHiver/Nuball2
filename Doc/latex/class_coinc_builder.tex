\doxysubsection{Detailed Description}
Coincidence builder. 

Use this class when you don\textquotesingle{}t have any particular time reference Add one hit after the other using \doxyref{Coinc\+Builder\+::build()}{p.}{class_coinc_builder_a6fbd7caa8ab6ac6bbf4d14eb094b6417} If two consecutive hits are within a time window, add the following hits This class will return true when the current hit is out of the time window

Using Builder\+::keep\+Singles(bool) method will make \doxyref{Coinc\+Builder\+::build()}{p.}{class_coinc_builder_a6fbd7caa8ab6ac6bbf4d14eb094b6417} return true for lonely hits as well
